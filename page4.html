HTML Tags Learned in Unit C2

  • <html> - The root element of an HTML document.
  • <head> - Contains meta-information about the document, such as the title and meta tags.
  • <title> - Specifies the title of the document, shown in the browser's title bar or tab.
  • <meta> - Provides metadata about the HTML document, such as author and character set.
  • <body> - Contains the content of an HTML document, such as text, images, and other elements.
  • <h1> to <h6> - Header tags, used to define headings, with <h1> being the highest level and <h6> the lowest.
  • <p> - Defines a paragraph of text.
  • <a> - Defines a hyperlink, used to link to another page or resource. It is an inline element. To locate the link, you need the href attribute.
  • <b> - This text will appear bold.
  • <strong> - This text will appear bold/strong.
  • <i> - This text will appear in italics.
  • <mark> - This text will appear highlighted/marked.
  • <small> - This text will be smaller.
  • <del> - This text will appear crossed out.
  • <ins> - This text will appear underlined.
  • <sub> - This text will appear as subscript.
  • <sup> - This text will appear as superscript.
  • <ol> - Creates an ordered list with numbers.
  • <ul> - Creates an unordered list with bullet points.
  • <p style="color: red"> - This paragraph will have red text.

Examples of hyperlink targets:

  • _blank: Opens in a new window/tab.
  • _top: Opens at the top of the page.
  • _self: Opens within the same iframe.
  • _parent: Opens within the parent of the current frame.
